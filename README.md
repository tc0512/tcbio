# tcbio
初中生的生物工具箱

## 安装
```bash
pip install https://ghproxy.net/https://github.com/tc0512/tcbio/releases/download/v2026.3.26/nbmath-2026.3.26-py3-none-any.whl
```

## 使用示例
```python
from tcbio import organism_and_cell as oac
from tcbio import helpers as h
oac.单目显微镜的结构() #启动一个单目显微镜结构的交互式程序
print(h.__version__()) #版本号
print(h.license()) #开源许可证
```

## 模块feature
- 简单易用
- 受众群体广，可供学生查询资料，部分功能也可供教师教学使用
